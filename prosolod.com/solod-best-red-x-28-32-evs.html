Солод BEST Red X 28-32 ЕВС

Мешок 50 кг
Производитель: BestMalz

Солод BEST Red X® был разработан для изготовления пива с красной цветностью. Его можно применять для 100% закладки — другие виды солода не понадобятся. Благодаря исключительной надежности и оптимальной способности к обработке, солод BEST Red X® может использоваться для варки пива с особо яркой огненной цветностью.

Благодаря своему полноценному вкусу и уникальному привлекательному цвету в сочетании с простотой пивоварения, солод BEST Red X® станет лучшим выбором для создания обширной номенклатуры новых сортов пива, например, красных пшеничных. Для янтарных и пшеничных сортов пива, эля, темного пива и всех сортов красного пива.
